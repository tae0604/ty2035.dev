**Aurum** is a virtual operating system (OS) project developed with a focus on fluid animations and a clean, minimalist user interface.
I built this on Entry, implementing smooth window dragging, live resizing, depth management (Z-index), and interactive widgets, pushing the limits of block-based programming.

## Key Features & Details
- **Glassmorphism UI**: A transparent, frosted-glass window design with smooth styling.
- **Multi-tasking**: Open, move, and interact with multiple active windows freely.
- **Drag & Drop**: Fluid movement and depth handling when dragging windows.
- **Interactive Apps**: Run custom widgets like a notepad, calculator, and custom system tools.
